Клеющая смесь для плитки CM11

Описание товара

Клеящая смесь предназначена для облицовки


Подробное описание товара

Клеящая смесь предназначена для облицовки бетонных, кирпичных, цементно-песчаных и цементно-известковых поверхностей плиткой с водопоглощением не менее 1% (плитки из керамики, фаянса и т.д. с размерами не больше чем 40 x 40 см).


Характеристики товара

  • для сухих и влажных помещений
  • высокая адгезия к основанию
  • устойчивая к сползанию c вертикальных поверхностей
  • водо- и морозостойкая
